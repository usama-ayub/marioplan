# Marioplan
marioplan web app is task management tool with using following Technology. 

1. ReactJS
2. Redux
3. React-Redux
4. React-Thunk
5. Redux-firebase
6. Firebase
7. Cloud Function
8. Firebase Hosting

Create Firebase project and Copy firebase connect key and paste to config/fbConfig.js file.
